## About the Project

This project is a frontend recreation of the Dropbox homepage created for learning and practice purposes.

The goal was to strengthen my understanding of responsive web design, CSS layouts, and building real-world landing pages from scratch without using any CSS framework.

The project recreates the visual design and layout of the Dropbox homepage while remaining fully responsive across mobile and desktop devices.



## Built With

* HTML5
* CSS3
* JavaScript (ES6)

## Features

* Fully responsive design
* Fixed navigation bar
* Mobile navigation menu with JavaScript
* Flexbox layouts
* CSS Grid layouts
* Responsive cards and content sections
* Horizontal testimonial slider
* Hover effects and transitions
* Embedded video content
* Multi-column responsive footer

## What I Learned

Through this project, I practiced:

* Structuring large webpages using semantic HTML
* Building responsive layouts with Flexbox and CSS Grid
* Creating mobile-first responsive designs
* Managing spacing, typography, and alignment across screen sizes
* Implementing a mobile navigation menu using JavaScript
* Working with images, videos, and interactive content
* Organizing CSS for large multi-section webpages
* Building a complete responsive landing page using only vanilla HTML, CSS, and JavaScript
* Recreating a production-style landing page from an existing website
* Deploying static websites using GitHub Pages
